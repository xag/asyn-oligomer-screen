# results

Pipeline artifacts. Most files here are regenerated from code +
`data/` — do not hand-edit.

<details>
<summary><b>Plain English</b></summary>

This folder holds the *outputs* of the analysis: every chart and table
shown in the main README, plus the relaxed 3D oligomer models and the
full candidate-molecule rankings. Everything in here is reproducible
from the code and the `data/` inputs — if you re-run the pipeline, these
files get rebuilt. Don't edit them by hand; if you want to change a
number, change the code or the inputs and re-run.
</details>

## `oligomers/`

Relaxed all-atom PDBs of the 11-topology ensemble built by
[oligomers/run_ensemble.py](../oligomers/run_ensemble.py), plus
`ensemble_summary.csv` (per-topology Stage-2 activity scores).
`fusco_parallel_3mer_core70-88_relaxed.pdb` is the reference target used
by the production sweep.

## `sweep/`

`<oligomer-stem>_sweep.csv` — one row per docked candidate molecule,
ranked by `delta_activity_gated`. Output of
[screen/sweep_oligomer.py](../screen/sweep_oligomer.py).

## `stage3/` (gitignored)

Per-pair docking artifacts (`*_docked.pdbqt`, `*_vina.log`,
`*_complex.pdb`, `*_report.json`). Regenerated by
[screen/stage3.py](../screen/stage3.py). Excluded from version control because the
sweep produces hundreds of files totalling many MB.
